## 6) `carrental.util` — 工具类（通用工具）

**一句话定位**：放置跨层次复用的工具方法，如密码哈希、输入校验、日期计算、CSV 导出等。

**职责**

* 提供静态工具方法（不可状态化），例如 `HashUtil.hash()` / `Validator.isEmail()` / `DateUtil.daysBetween()`
* 不依赖具体业务（不引用 DAO/Service/UI），可被任意包调用
* 可包含导出/导入工具（CSVExporter）

**常见类 / 方法**

* `HashUtil.hash(String)`、`HashUtil.verify(String, hash)`
* `Validator.isNotEmpty(String)`、`Validator.isValidPhone(String)`
* `DateUtil.toLocalDate(String)`、`DateUtil.daysBetween(start,end)`
* `CSVExporter.export(List<?> data, Path file)`

**注意事项**

* 保持无状态（静态方法为主）以方便测试
* 对外暴露的方法做好输入校验与异常说明

**package-info.java 模板**

```java
/**
 * carrental.util
 *
 * 说明：
 * 通用工具类包，包含密码哈希、字段校验、日期工具、CSV 导出等功能。
 * 这些工具应为无状态静态方法，可被多个层（service/ui/dao）复用。
 *
 * 设计要点：
 * - 避免与具体业务耦合（不要引用 dao/service 内部类）
 * - 方法应清晰标注异常及边界条件（如 null 处理）
 *
 * 常见类：
 * - HashUtil, Validator, DateUtil, CSVExporter
 */
package carrental.util;
```
